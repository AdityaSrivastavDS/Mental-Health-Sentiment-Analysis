# Mental Health Sentiment Analysis 🌱

This project involves sentiment analysis of over 51,000 records, aimed at identifying and categorizing various mental health conditions based on user statements. The dataset has been thoroughly cleaned, with preprocessing steps applied to ensure 99% accuracy. The project delivers valuable insights into mental health trends and provides a data-driven framework for further analysis and interventions.

## 📊 Key Features:
- **Over 51,000 cleaned records** for analysis with 99% data accuracy.
- **Seven mental health conditions** categorized, including Normal, Depression, Suicidal, Anxiety, Bipolar, Stress, and Personality Disorder.
- **Advanced preprocessing** that eliminated 100% duplicates, handled missing values, and standardized text lengths.
- **Exploratory Data Analysis (EDA)** with actionable insights into mental health trends.
- **Visualizations** including word clouds and sentiment distributions.

## 📑 Installation and Setup:
1. Clone this repository:
   ```bash
   git clone https://github.com/AdityaSrivastavDS/Mental-Health-Sentiment-Analysis
   cd Mental-Health-Sentiment-Analysis
   ```

2. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the analysis:

   ```bash
   jupyter Mental Sentiment Analysis.ipnyb
   ```

## 🛠️ Preprocessing:
- **Data Cleaning:** Duplicates were removed and missing values handled.
- **Text Standardization:** Text lengths were standardized, and all entries were normalized.
- **Tokenization:** Statements were tokenized for easier analysis.

## 📈 Data Visualizations:
